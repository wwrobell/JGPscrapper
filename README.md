# JGPscrapper
JGP data mining tool

## Purpose
* Explore and process data from NFZ ‘JGP directory’ (https://prog.nfz.gov.pl/app-jgp/KatalogJGP.aspx)

Scripts were developed to calculate number of annual ECG diagnostics in Poland.

## Requirements
* Python 3.6
* Installed packages listed in requirements.txt. To install requirements run:
 
 Linux:
 ```bash
 pip install -r requirements.txt
 ```
 
 Windows:
 1. Move to the directory with the used interpreter python.exe.
 2. Run:
 ```bash
 python.exe -m pip install -r path_to_requirements.txt
 ```
